## What was the Frutiger Aero Era?

**Frutiger Aero** was the dominant design language across technology, media, and architecture roughly between 2004 and 2013. It has a really cool community.
You can read more about it here - [The Frutiger Aero Archive](https://frutigeraeroarchive.org/)

This was my primary source for inspiration, wallpapers, information and other resources.
